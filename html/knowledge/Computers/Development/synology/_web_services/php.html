Php verification

Install php 
dnf -y install php.x86_64
Edit the php.ini file 
sed -ie s/\;date\.timezone.*/date.timezone=America\\/Chicago/ /etc/php.ini
Create the phpinfo.php page 
vi /data/www/html/unencrypted/private/phpinfo.php

i

<?php
phpinfo();
?>
Create the unencrypted,public, hello world pages 
vi /data/www/html/unencrypted/public.php

<html>
<head>
<title>Unencrypted Public</title>
</head>
<body>
<?php
phpinfo();
?> 
</body>
</html>
Create the unencrypted,private page 
vi /data/www/html/unencrypted/private/private.php

<html>
<head>
<title>Unencrypted Private</title>
</head>
<body>
<?php
phpinfo();
?> 
</body>
</html>
Create the encrypted,public page 
vi /data/www/html/ssl/public.php

<html>
<head>
<title>Encrypted Public</title>
</head>
<body>
<?php
phpinfo();
?> 
</body>
</html>
Create the encrypted,private page 
vi /data/www/html/ssl/private/private.php

<html>
<head>
<title>Encrypted Private</title>
</head>
<body>
<?php
phpinfo();
?> 
</body>
</html>
Change the ownership of the apache directory 
chown -R apache:apache  /data/www/html
chmod -R 700 /data/www/html

systemctl restart httpd
From the index page, check the "Unencrypted Public PHP Page", "Unencrypted Private PHP Page", "Encrypted Public PHP Page" and the "Encrypted Private PHP Page" 
http://$_CURRENT_IP/private
