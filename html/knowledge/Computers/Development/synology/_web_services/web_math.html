JSMath - A Javascript Library for Mathematic Typesetting in Web Pages

Install jsMath 
unzip $_FILES/jsMath-3.6e.zip -d /data/www/javascript/

unzip $_FILES/jsMath-fonts-1.3.zip -d /data/www/javascript/

cd /data/www/javascript/

mv jsMath/fonts jsMath-3.6e/

rmdir jsMath

ln -s jsMath-3.6e jsMath
Create test page 
vi /data/www/html/unencrypted/jstest.html
