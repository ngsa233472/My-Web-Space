Install fortune

Add the fortune program and the Calvin and Hobbes fortunes Install the fortune rpm 
dnf -y install fortune-mod
Install the Calvin and Hobbes data file 
tar zxvf $_FILES/fortune-mod-calvin-0.1.1.tar.gz -C /usr/src/

cd  /usr/src/fortune-mod-calvin-0.1.1/

sed -ie s/fortunes/fortune/g Makefile

make install

fortune /usr/share/games/fortune/calvin

Note this step

/usr/local/bin/note.sh fortune
