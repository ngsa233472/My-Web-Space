Gcc and G++ Installation, Configuration, and Testing

Install GCC 
dnf -y install gcc.x86_64
Install G++ 
dnf -y install gcc-c++.x86_64
Install C Development Tools and Libraries 
dnf -y group install "C Development Tools and Libraries"
Create the testfile 
mkdir /data/src/g++
