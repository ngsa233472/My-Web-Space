Java SDK Installation, Configuration, and Testing

Get the file and install the rpm 
dnf -y install java-11-openjdk.x86_64
dnf -y install java-11-openjdk-devel.x86_64
Check the installation 
java -version

You should get something like this

java version "1.8.0_112"
Java(TM) SE Runtime Environment (build 1.8.0_112-b15)
Java HotSpot(TM) 64-Bit Server VM (build 25.112-b15, mixed mode)
Create the java environment file 
cat << CONFIG > /etc/profile.d/java.sh
#!/bin/bash

JAVA_HOME=/usr/lib/jvm/java-11-openjdk-11.0.3.7-5.fc30.x86_64
CLASSPATH=.

export JAVA_HOME
export CLASSPATH

CONFIG

. /etc/profile.d/java.sh
Create and run Test Program 
mkdir -p /usr/src/java 

cat << REQUIRED > /usr/src/java/HelloWorld.java
/*************************************************************************
*  Compilation:  javac HelloWorld.java
