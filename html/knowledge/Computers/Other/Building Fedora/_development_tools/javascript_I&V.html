Javascript Installation, Configuration, and Testing

Create the Javascript directory 
mkdir -p /data/www/javascript

vi /etc/httpd/conf.d/javascript.conf

dG
i

Alias /js /data/www/javascript

<Directory "/data/www/javascript">
        Options Indexes FollowSymLinks
        Require all granted
        AllowOverride None
</Directory>
Create the testfile 
cat << REQUIRED > /data/www/html/unencrypted/javascript.html
<!DOCTYPE html>
<html>
<body>

<h1>My First JavaScript</h1>

<button type="button"
onclick="document.getElementById('demo').innerHTML = Date()">
Click me to display Date and Time.</button>

<p id="demo"></p>

</body>
</html>

REQUIRED
Fix the ownership of the apache directory 
chown -R apache:apache /data/www
find /data/www -type d -exec chmod 755 {} \;
find /data/www -type f -exec chmod 644 {} \;
Restart the daemon 
/bin/systemctl restart httpd
Check the daemon is running 
/bin/systemctl status httpd
From the index page, check the "Javascript Test Page" 
http://$_CURRENT_IP/private

/usr/local/bin/note.sh javascript_IVandV
