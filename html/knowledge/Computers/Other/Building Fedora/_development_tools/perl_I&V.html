Create the source directory 
mkdir /data/src/perl
Create the testfile 
cd /data/src/perl
vi helloworld.pl

#!/usr/bin/perl
print "Hi there!\n";

chmod 755 helloworld.pl
Run the testfile 
./helloworld.pl

You should get something like this

Hi there!
