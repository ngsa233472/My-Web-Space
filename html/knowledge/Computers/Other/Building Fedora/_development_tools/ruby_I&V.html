Ruby Installation, Configuration, and Testing

Install Ruby 
dnf -y install ruby.x86_64
Check the installation 
ruby -v

You should get something like this

ruby 2.6.3p62 (2019-04-16 revision 67580) [x86_64-linux]
Create the source directory 
mkdir /data/src/ruby
Create the testfile 
cd /data/src/ruby

vi helloworld.rb

#!/usr/bin/ruby
puts "Hello World!";
puts "Welcome to Ruby Programming"

chmod 755 helloworld.rb
Run the testfile 
./helloworld.rb

You should get something like this

Hello World!
Welcome to Ruby Programming

Note this step

/usr/local/bin/note.sh rubyIandV
