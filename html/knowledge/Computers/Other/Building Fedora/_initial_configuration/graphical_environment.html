Install Graphical Environment

Install X and Gnome Shell 
dnf -y group install base-x
dnf -y install gnome-shell
dnf -y install gnome-terminal.x86_64
Check the graphical environment 
systemctl set-default graphical.target
systemctl enable gdm
Reboot the system and check in VMWare
Set back to text environment 
systemctl set-default multi-user.target
Reboot the system and check in VMWare
Note this step 
/usr/local/bin/note.sh install_graphical_environment
