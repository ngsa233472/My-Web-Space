Create an admin users account

To configure accounts that have enhanced access to the system such as sudo to root Add user to sudoers 
USER='nick'
sed -i -e '/^root\tALL=(ALL)/a\'$USER'\tALL=(ALL)\tNOPASSWD:ALL' /etc/sudoers

NOTE TYPES OF DATA THAT IS GOING HERE/MAIL,FILES,PERSONAL WEB PAGES,LINKS

Note this step

/usr/local/bin/note.sh add_admin_users
Exit root 
exit
