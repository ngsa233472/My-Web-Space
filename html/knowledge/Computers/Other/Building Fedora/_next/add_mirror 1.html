Mirrors

Purpose To add a mirrored data directory to the system Prerequisites Two extra hard disk installed as /dev/sdb and /dev/sdc Login as root Create a partition on /dev/sdc 
fdisk /dev/sdc
Add /dev/sdc to the mirror 
mdadm --manage /dev/md0 --add /dev/sdc1
Check the status of the mirror 
cat /proc/mdstat

Note this step

/usr/local/bin/note.sh added_mirror
