Apache Ant

Purpose A Java build tool like Make. Used for building java related programs
Prerequisites install java
Login as root
Install Ant 
unzip $_FILES/apache-ant-1.8.2-bin.zip -d /opt

Moving quickly

cd /opt
ln -s apache-ant-1.8.2 apache-ant
echo "export ANT_HOME=/opt/apache-ant" >> /etc/profile.d/custom.sh
echo "export PATH=\${PATH}:\${ANT_HOME}/bin" >> /etc/profile.d/custom.sh
Note this step 
touch /install/install_apache_ant.$INSTALL_VERSION
