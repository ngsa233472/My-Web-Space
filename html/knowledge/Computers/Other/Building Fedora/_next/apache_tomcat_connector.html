Configure Tomcat-Connector

Allows the connection of the Apache server (1st Tier) to the Tomcat instance (2nd Tier) Prerequisites
Java
Tomcat
Create a apache connector conf file 
vi /etc/httpd/conf.d/tomcat.conf


ProxyPass /tomcat ajp://$_CURRENT_IP:8009
ProxyPassReverse /tomcat http://$_CURRENT_IP
Restart both the web and application server 
systemctl restart tomcat

systemctl restart httpd
from the browser, run the servlet http://$_CURRENT_IP/tomcat/SimpleServlet/simple-servlet

Note this step

/usr/local/bin/note.sh apache_tomcat_connector
