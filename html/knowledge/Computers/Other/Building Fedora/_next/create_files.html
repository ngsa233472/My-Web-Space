The idea is to create a repository that you can sftp over to the server after you have built the $_FILES directory. In this manner, you can get large files even if the system is off line. This is an easy way to install files that are not installed via dnf

On a machine that you can sftp from create a set of files that you can bring down into /data/FILES on the newly built server
