How do I change the IP Address of the target in the documentation? Can I use this to install to a different machine?
Yes, copy all the bundled files to a machine that you can do a cross file replacement (sed, ActiveState Komodo),
1. Replace the main IP in all the files ($_CURRENT_IP, XXX.XXX.XXX.XXX), where the X's represent the desired IP.
2. Now change the first three octets in all files (192.168.122, XXX.XXX.XXX), where the X's represent the desired IP. This will modify file records that use just the netmask or the first three octets in common, like DNS resource records.
