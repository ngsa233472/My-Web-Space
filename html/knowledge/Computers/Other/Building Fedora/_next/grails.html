Apache Ant

Purpose A Java build tool like Make. Used for building java related programs Prerequisites
Java
Login as root
Install Grails 
cp /data/FILES/grails-2.2.3.zip /opt
cd /opt
unzip grails-2.2.3.zip
Link current version to generic version 
cd /opt
ln -s grails-2.2.3 grails
Update paths 
echo "export GRAILS_HOME=/opt/grails" >> /etc/profile.d/grails.sh
echo "export PATH=\${PATH}:\${GRAILS_HOME}/bin" >> /etc/profile.d/grails.sh
Test Grails 
grails -version

Note this step

/usr/local/bin/note.sh grails
