While testing build Iona inside a virtual machine

Select new virtual machine from VM Workstation
Select typical configuration
Select install o/s later
Select Linux and Fedora 64-bit
Name installation "IONA"
Set disk size to 20GB
Select customize hardware
Change drive to us the Fedora ISO
Change Network to be bridged
Power on virtual machine
Skip the media test
Select the language
Select the keyboard
Select basic storage devices
Discard existing data
Set the hostname
Setup the network
Set the timezone to Chicago
Set the password
Use the entire disk
Choose "Applications:Graphical Desktop" installation and select customize now
In Applications:Editors, turn on Emacs
In Applications:Games and Entertainment, turn off everything but "Fortune"
Deselect Applications:Graphical Internet, Applications:Graphics, and Applications:Office Productivity"
In Applications:Sound and Video, turn off everything except "k3b" and "wodim"
In Development select Development Libraries, Development Tools, Fedora Eclipse, Fedora Packager, GNOME Software Development, Web Development and X Software Development
In Servers:Choose DNS Name Server, Mail Server, MySQL Database, PostgreSQL Database, Printing Support, Web Server, and Windows File Server
Start build
