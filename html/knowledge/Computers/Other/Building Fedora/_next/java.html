Java SDK Installation, Configuration, and Testing

Purpose Install the JAVA Language development kit for use by other programs and development of java required Prerequisites None Login as root Remove the old JAVA installation links
rm /usr/bin/java /usr/bin/javac /usr/bin/javadoc /usr/bin/javah /usr/bin/javaws
Copy the JAVA binary to the home directory the old JAVA installation links
cp $_FILES/jdk-6u29-linux-x64-rpm.bin .
Make the binary executable
chmod 755 jdk-6u29-linux-x64-rpm.bin
Run the binary
./jdk-6u29-linux-x64-rpm.bin
Create the new JAVA installation links
ln -s /usr/java/latest/bin/javac /usr/bin/javac && ln -s /usr/java/latest/bin/javadoc /usr/bin/javadoc && ln -s /usr/java/latest/bin/javah /usr/bin/javah && ln -s /usr/java/latest/bin/javaws /usr/bin/javaws
Install the initial CLASSPATH
echo -e "export JAVA_HOME=/usr/java/jdk1.6.0_29">>/etc/profile.d/custom.sh
