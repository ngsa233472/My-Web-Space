Java SDK Installation and Verification

Login with Putty and continue configuration

Install the JDK

Install the runtime 
yum install -y java-1.8.0-openjdk.x86_64
Install the development rpm 
yum install -y java-1.8.0-openjdk-devel.x86_64
Check the installation 
java -version
You should get something like this 
openjdk version "1.8.0_25"
OpenJDK Runtime Environment (build 1.8.0_25-b18)
OpenJDK 64-Bit Server VM (build 25.25-b02, mixed mode)

Test the JDK

Create and run Test Program 
mkdir -p /usr/src/java
