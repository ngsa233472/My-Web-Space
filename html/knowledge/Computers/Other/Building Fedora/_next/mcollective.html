MCollective

Pre-requisite
Install Java
MCollective is a distributed shell. It uses ActiveMQ to queue and distribute commands. Install MCollective Server 
dnf -y install mcollective
Install MCollective Client 
dnf -y install mcollective-client
Install Install ActiveMQ from the local repository 
dnf -y localinstall $_FILES/tanukiwrapper-3.5.9-1.el5.x86_64.rpm
dnf -y localinstall $_FILES/activemq-5.5.0-1.el5.noarch.rpm
dnf -y localinstall $_FILES/activemq-info-provider-5.5.0-1.el5.noarch.rpm
Start ActiveMQ 
/etc/init.d/activemq start
Verify ActiveMQ running 
ps auxw|grep java

netstat -plt --numeric-ports | grep 61613
Check the ActiveMQ admin page with the password admin/admin 
ps auxw|grep java
http://192.168.122.10:8161/admin/
Start MCollective 
systemctl start mcollective
Test MCollective 
mco ping

Note this step

/usr/local/bin/note.sh mcollective
