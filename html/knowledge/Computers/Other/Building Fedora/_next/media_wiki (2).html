Login as root Install APC via yum 
yum -y install php-ZendFramework-Cache-Backend-Apc.noarch
Restart Apache 
/bin/systemctl restart httpd
Install Mediawiki via yum 
yum -y install mediawiki
Edit mediawiki config file 
vi /etc/httpd/conf.d/mediawiki.conf

ln -s /usr/share/mediawiki /data/www/html/wiki

/bin/systemctl restart httpd
Check basic page in browser Click on the "set up the wiki" link
Follow the instructions on the page to continue the mediawiki setup
Move the config file 
mv /data/www/html/wiki/config/LocalSettings.php /data/www/html/wiki
