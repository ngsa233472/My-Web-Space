Sendmail Milter

a. Purpose A milter (portmanteau for mail filter), is an API enables third-party programs to access mail messages as they are being processed by the Mail Transfer Agent (MTA). Login as root c. Check Sendmail for Milter

Verify that sendmail was compiled and installed with the MILTER option

sendmail -bt -d0.1 < /dev/null | grep -i milter
You should get something like this!: 
		MATCHGECOS MILTER MIME7TO8 MIME8TO7 NAMED_BIND NETINET NETINET6

Note this step

/usr/local/bin/note.sh milter
