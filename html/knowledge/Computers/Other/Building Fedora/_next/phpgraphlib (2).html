Install and configure PHPGraphLib

Simple method for graphically displaying data in html pages

Login as root 
yum -y install php-gd
Cd to html directory 
cd /data/www/html/unencrypted/private/
Get the phpgraph lib 
wget http://www.ebrueggeman.com/sites/www.ebrueggeman.com/files/phpgraphlib_v2.30.zip
Unpack the phpgraph lib 
unzip phpgraphlib_v2.30.zip
rm phpgraphlib_v2.30.zip
Create the graph_bar.php file 
vi graph_bar.php

i

<?php
include("phpgraphlib.php");
$graph = new PHPGraphLib(400,300);
$data = array("Alex"=>99, "Mary"=>98, "Joan"=>70, "Ed"=>90);
$graph->addData($data);
$graph->setTitle("Test Scores");
$graph->setTextColor("blue");
$graph->createGraph();
?>
Create the graph_bar.html file 
vi graph_bar.html

i

<html>
<h3>This is where I want to display my graph
<img src="graph_bar.php" />
<html>
Restart System 
reboot
Check demo graph page in browser

Note this step

/usr/local/bin/note.sh phpgraphlib
