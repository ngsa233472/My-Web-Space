Php verification

Login as root Install php-fpm 
yum -y install php-fpm.x86_64
Edit the php.ini file 
sed -ie s/\;date\.timezone.*/date.timezone=America\\/Chicago/ /etc/php.ini
Start php-fpm 
systemctl enable php-fpm

systemctl start php-fpm
Create the phpinfo.php page 
vi /data/www/html/unencrypted/private/phpinfo.php

i

<?php
phpinfo();
?>
Change the ownership of the apache directory 
chown apache:apache  /data/www/html/unencrypted/private/phpinfo.php
chmod 700  /data/www/html/unencrypted/private/phpinfo.php

Note this step

/usr/local/bin/note.sh phpinfo
