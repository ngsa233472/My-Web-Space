Install phpldapadmin

Login as root Install phpldapAdmin 
yum install -y phpldapadmin
Add the 192.168.122.0/8 network to the access list 
sed -ie '/\<Directory>/i\ \ Allow from 192.168.122.0/8' /etc/httpd/conf.d/phpldapadmin.conf
Restart apache 
/bin/systemctl restart httpd
Check phpldapAdmin page in browser
