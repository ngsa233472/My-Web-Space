Proxy Server/Client Configuration

Allows secure web surfing via SSH and Socks Configure browser Configure the browser to use the proxy. using Firefox as an example. Open up preferences, and go to Advanced/Network and click on Settings:

In advanced settings choose "Manual Proxy Configuration" and "socks v4". In the "Socks Host" field put "localhost" and in the port for the socks host put 8080, and in the "No proxy for" out "localhost,127.0.0.1"



The last thing to do on the browser, is to change where it gets it DNS information from. in the URI type in "about:config". Search for the the entry "network.proxy.socks_remote_dns" and set it to true.
Open a terminal on the client
Nicks:~ nick$ ssh -C2qTnN -D 8080 grayshouse.dyndns.org
At this point all dns and page requests will be routed through the proxy server. This will not affect any other protocol or program.
