So what is a rebuild?

One of the advantages of these pages is, they have been tested, many many times. Whenever I think there is a change that warrants it, I will rebuild, using the instructions completely from the beginning.

Below are steps needed to save data from a system, so that it can be rebuilt

Make a backup target
Save Davical Database (postgres)
Save WikiMedia data (MySQL+)
Save Wordpress data (MySQL+)
Saving data to safe location
Copy the install directory to the backup
Save the backup to media
