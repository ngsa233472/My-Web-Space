SaslAuthd Configuration

Login as root Set up SASL Check the Sendmail.conf file 
cat /etc/sasl2/Sendmail.conf

You should get this!:

pwcheck_method:saslauthd
Configure startup 
/bin/systemctl enable saslauthd

/bin/systemctl start saslauthd
Test saslauthd 
testsaslauthd -u testuser -p rootMe55 -s smtp

You should get something like this!:

0: OK "Success."

Note this step

/usr/local/bin/note.sh saslauthd
