Comments and Some Typically Used Statements

Change to the mail directory 
cd /etc/mail
Open sendmail.mc 
vi /etc/mail/sendmail.mc
Comments in Sendmail Comments come in two forms. Those that get added to the resulting .cf file and those that dont

Lines in the sendmail.mc file that begin with the # character are not parsed by m4, and will by default be output directly into the sendmail.cf file.

To allow comments in your sendmail.mc that are not placed into the sendmail.cf, you can use the m4 divert and dnl tokens.

divert(-1) will cause all output to cease. divert(0) will cause output to be restored to the default. Any output generated by lines between these will be discarded.
To achieve the same result for a single line, you can use the dnl token.

sed -i -e "1i\divert\\(-1\)\n\
Nick\\'s Master Sendmail\n\
divert\\(0\)" /etc/mail/sendmail.mc
Common Statements 
sed -i -e "\$adnl\nMAILER\\(\\\`local\\'\\)" sendmail.mc

sed -i -e "\$aMAILER\\(\\\`smtp\\'\\)" sendmail.mc

sed -i -e "/OSTYPE/i \VERSIONID\(\`\@\(#)Nick\\'s Sendmail File \$Revision: 3.0 \$\\'\\'\)dnl" sendmail.mc
