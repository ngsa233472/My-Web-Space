Install VNC Server

Purpose Login as root Install VNC 
yum install -y tigervnc
yum install -y tigervnc-server
Modify /etc/sysconfig/vncservers 
vi /etc/sysconfig/vncservers

dG
i

VNCSERVERS="1:root"
VNCSERVERARGS[1]="-geometry 1400x800 -nolisten tcp -localhost"
Start vnc on boot 
chkconfig vncserver on
Cd into root 
cd
Create the users vncdir 
mkdir .vnc
Modify .vnc/xstartup 
vi .vnc/xstartup

dG
i

vncconfig -iconic &
unset SESSION_MANAGER
exec /etc/X11/xinit/xinitrc
Modify mode bits on the xstartup 
chmod 755 .vnc/xstartup
Create a password for root 
vncpasswd
Configure startup 
chkconfig vncserver on

service vncserver start
Test server 
vncviewer localhost:5901

Note this step

/usr/local/bin/note.sh install_vnc
Exit root 
exit
