Enable a Method of Including Mathematics in Web Pages

Login as root Download jsMath-3.6e.zip amd the jsMath Image Fonts
Link to jsMath sourceforge site
Install jsMath 
unzip $_FILES/jsMath-3.6e.zip -d /data/www/html/unencrypted/

unzip $_FILES/jsMath-fonts-1.3.zip -d /data/www/html/unencrypted/

cd /data/www/html/unencrypted

mv jsMath/fonts jsMath-3.6e/

rmdir jsMath

ln -s jsMath-3.6e jsMath

chown -R apache:apache jsMath*
Create test page 
vi /data/www/html/unencrypted/jstest.html
