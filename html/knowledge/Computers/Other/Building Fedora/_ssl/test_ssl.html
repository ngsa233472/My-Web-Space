Testing OpenSSL Certificates

To test the newly generated certificates. Testing the CA certificate 
openssl verify -CAfile /etc/pki/CA/certs/CA.crt /etc/pki/CA/certs/CA.crt
The following result is testing the self signed CA against itself. 
/etc/pki/CA/certs/CA.crt: OK
Testing the Server certificate 
openssl verify -CAfile /etc/pki/CA/certs/CA.crt /etc/pki/tls/certs/server.crt
The following should show that the cert is valid against the self signed CA 
/etc/pki/tls/certs/server.crt: OK
Note this step 
/usr/local/bin/note.sh test_openssl
