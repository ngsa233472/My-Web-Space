Academics/Old School Notes/Exponents

Def:
A^p = e^{p \ ln A} \ \ if A > 0
Laws:
x^m \cdot x^n = x^{m+n}
{x^m \over x^n} = x^{m-n}
(x^m)^n = x^{m \cdot n}
(xy)^m = x^m y^m
({x \over y})^m = {x^m \over y^m}
x^{-m} = {1 \over x^m}
({x \over y})^{-m} = ({y \over x})^m
Derivatives:
D_x ( e^x ) = e^x
D_x ( e^f ) = e^f{df \over dx}
Limits:
{lim \over x \rightarrow \infty} e^{x} = \infty
{lim \over x \rightarrow \infty} e^{-x} = 0
Integrals:
\int {1 \over x} dx = ln \mid x \mid + C
\int {1 \over {ax + b}} dx = {1 \over a}ln \mid ax+b \mid + C \ \ \ where \ \ a \ne 0
