rules of operations on limits
\begin{align} 1. {lim \over x \rightarrow n} k = k \\ 2.{lim \over x \rightarrow 0} 5 = 5 \\ 3. {lim \over x \rightarrow a} a = a \\ 4. {lim \over x \rightarrow 4} 4 = 4\\ as \ x \rightarrow 0 \ or \ x \rightarrow \infty\\ lim \ 4x + 3x = lim \ 4x + lim \ 3x \\ lim \ 2g3y = lim \ 2g * lim \ 3y \\ lim \ {2g \over 3y} = {{lim \ 2g} \over {lim \ 3y}} \\ lim \ {\root n\of{ f(x)}} = \root n\of {lim \ f(x)} \\ lim {P(x) \over {x \rightarrow a}} = P(a) \\ \end{align}
Special limits
\begin{align} {0 \over 0} \ Use L'hopital's rule
{{\infty \over \infty}, \ \infty - \infty, \ or \ \ 0 * \infty} \ : Convert to {0 \over 0} \ Use L'hopital's rule \end{align}
