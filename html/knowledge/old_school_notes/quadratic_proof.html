Academics/Old School Notes/Proof of the Quadratic

ax^2 + bx +c = 0           Given
ax^2 + bx = -c               Subtract c from both sides
x^2 + {b \over a}x = {-c \over a}               Divide by a
(x^2 + {b \over 2a}x + {b \over 2a}x + ({b \over 2a})^2 ) - ({b \over 2a})^2 = {-c \over a}               Complete the square
(x + {b \over 2a})^2 - ({b \over 2a})^2 = {-c \over a}
(x + {b \over 2a})^2 = ({b \over 2a})^2 + {-c \over a}
(x + {b \over 2a})^2 = {b^2 \over 4a^2} + {-c \over a}
(x + {b \over 2a})^2 = {b^2 \over 4a^2} + ({4a \over 4a}{-c \over a})
(x + {b \over 2a})^2 = {b^2 \over 4a^2} + {-4ac \over 4a^2}
(x + {b \over 2a})^2 = {b^2 - 4ac \over 4a^2}
x + {b \over 2a} = \pm \sqrt {{b^2 - 4ac \over 4a^2}}
x = - {b \over 2a} \pm \sqrt {{b^2 - 4ac \over 4a^2}}
x = - {b \over 2a} \pm {\sqrt {b^2 - 4ac} \over 2a}
x = {-b \pm \sqrt {b^2 - 4ac} \over 2a}
